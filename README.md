# 🏡 Dream Reality Property

Dream Reality Property is a modern, responsive real estate web application designed to help users browse, filter, and find their dream homes. This project showcases clean user interface design, smooth navigation, and dynamic property listings.

## ✨ Features
* **Property Search & Filters:** Filter properties by location, price, property type (apartment, villa, condo), and size.
* **Interactive Image Galleries:** View high-quality images of featured properties listed by Dream Reality Property.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewports.
* **Contact Agent Form:** A dedicated form for users to book tours or request more information on listings.

* **Frontend:** HTML5, CSS3, JavaScript
* **Icons & Fonts:** FontAwesome / Google Fonts
